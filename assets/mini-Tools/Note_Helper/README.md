# 长图制作工具

一个纯前端本地工具，用于把多张图片合并成一张竖向长图，并可添加文字水印或图片水印。

## 使用方式

直接用浏览器打开 `index.html`。

1. 点击上传区，或把图片拖入上传区。
2. 在「图片队列」中调整顺序或删除图片。
3. 按需选择纯色或彩色花边框。
4. 按需启用文字水印或图片水印，并设置位置、透明度、边距。
5. 点击「生成预览」。
6. 点击「下载 PNG」导出长图。

文字水印支持设置数量、颜色、字号和角度；图片水印支持设置数量和图片宽度。数量大于 1 时，水印会沿长图纵向分布；在预览画布中拖动水印，可调整自定义位置。

选择图片水印并上传水印图片后，右侧会显示水印图片缩略图；如果已经上传了待拼接图片，中间画布会自动刷新并显示水印效果。

## 清晰度说明

工具会按每张图片的原始像素绘制到 Canvas，不主动缩放或压缩图片。最终默认导出 PNG，避免 JPEG 质量损失。

如果多张图片宽度不同，画布宽度会采用最大图片宽度，较窄的图片会居中放置，原图尺寸保持不变。

## 宽度策略

工具提供 4 种拼接方式：

- 自动统一宽度：只缩小。推荐用于宽度不一致的图片，会把所有图片统一到最窄图片的宽度，避免放大小图导致模糊。
- 原始尺寸居中。完全保留原始像素，不缩放；宽度不同的图片会居中显示。
- 强制统一为最大宽度。所有图片统一到最大图片宽度，小图会被放大，版面整齐但可能变糊。
- 自定义宽度。按指定宽度等比缩放，适合固定输出规格。

严格意义上，不缩放才是像素级无损。实际做长图时，更推荐“只缩小不放大”：版面统一，细节损失最少，且不会把低分辨率图片硬拉糊。

## 边框样式

边框会绘制进最终导出的 PNG 中，支持：

- 无边框
- 纯色边框
- 彩色渐变
- 彩色圆点花边
- 糖果条纹花边

可以设置边框宽度；纯色边框还可以单独设置颜色。
