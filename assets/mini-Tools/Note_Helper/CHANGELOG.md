# Changelog

所有版本更新内容记录在这里。

## v0.4.0 - 2026-06-08

- 文字水印支持在预览画布中拖动位置。
- 文字水印支持设置数量，多个水印会沿长图纵向分布。
- 文字水印支持设置角度，范围为 -180 到 180。
- 文字水印和图片水印分别保存自定义拖动位置，切换类型时互不覆盖。

## v0.3.0 - 2026-06-08

- 新增边框设置。
- 支持无边框、纯色边框、彩色渐变边框、彩色圆点花边、糖果条纹花边。
- 支持设置边框宽度。
- 纯色边框支持自定义颜色。
- 边框会绘制进最终导出的 PNG，不只是页面预览装饰。

## v0.2.0 - 2026-06-08

- 图片水印上传后在右侧显示缩略图和尺寸信息。
- 图片水印上传后自动启用水印，并刷新长图预览。
- 图片水印支持在预览画布中拖动位置。
- 图片水印支持设置数量，多个水印会沿长图纵向分布。

## v0.1.0 - 2026-06-08

- 新增长图制作工具初版。
- 支持多图片上传、拖拽上传、队列排序、删除和清空。
- 支持按图片原始像素竖向拼接。
- 支持不同宽度图片的拼接宽度策略：
  - 自动统一宽度：只缩小。
  - 原始尺寸居中。
  - 强制统一为最大宽度。
  - 自定义宽度。
- 支持文字水印和图片水印。
- 支持设置水印位置、透明度和边距。
- 支持生成预览并导出 PNG。
